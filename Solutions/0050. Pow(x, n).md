## 解法1: 常规迭代
TC: O(N) SC:O(1)
```
class Solution:
    def myPow(self, x: float, n: int) -> float:
        res = 1
        if n > 0:
            for i in range(n):
                res *= x
        else:
            for i in range(-n):
                res *= x
            res = 1/ res
        return res
```
