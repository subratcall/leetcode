做这题的时候，本能地想到把结果储存在一个两层list嵌套的矩阵中。但是这样就需要不停地移动index，增加了很大的难度。

事实上，这题就是不停地在行之间依次切换，每一行的结果用string保存即可。这样就降低了难度。

TC： O(n) SC:O(n)
```
class Solution:
    def convert(self, s: str, numRows: int) -> str:
        if numRows == 1:
            return s 
        
        res = [""] * numRows
        row = 0
        down = True
        for i in range(len(s)):
            # 先填数
            res[row] += s[i]
            
            # 再移动光标
            if i % (2 * numRows - 2) == numRows - 1:
                down = False
            if i % (2 * numRows - 2) == 0:
                down = True
                
            if down:
                row += 1
            else:
                row -= 1
                
        return "".join(res)

```
