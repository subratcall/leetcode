## 解法1: Brute Force
枚举start/end，并求和，检查是否为k

TC: O(n^3) SC:O(1)


## 解法2: Cumulative Sum

求出前缀和之后，套两层for循环来枚举。

TC: O(n^2) SC:O(n)
```
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        
        prefix = [0] * (len(nums) + 1)
        total = 0
        count = 0
        
        for i in range(0, len(nums)):
            total += nums[i]
            prefix[i + 1] = total
            
        for i in range(len(prefix)):
            for j in range(i + 1, len(prefix)):
                if prefix[j] - prefix[i] == k:
                    count += 1
        return count
```

思考怎么优化这两层for循环。这两层for循环找两数之差的感觉，很像2Sum。

因此可以用同样的办法，用空间换时间。而且现在空间复杂度已经有O(n)了，实际没影响。

```
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        
        prefix = [0] * (len(nums) + 1)
        total = 0
        count = 0
        
        for i in range(0, len(nums)):
            total += nums[i]
            prefix[i + 1] = total
        
        from collections import defaultdict
        occu = defaultdict(int)
        
        for num in prefix:
            count += occu[num - k]
            occu[num] += 1
                        
        return count
```
