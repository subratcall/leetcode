## 解法2: 同向双指针
注意这里应该是nums[i+1]=nums[j]。因为第一个元素肯定是可以不变的。
```
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        if not nums:
            return len(nums)
        j = 1
        for i in range(len(nums)):
            j = max(i+1, j)
            while j < len(nums) and nums[j] == nums[i]:
                j += 1
            if j == len(nums):
                break
            nums[i + 1] = nums[j]
        return i + 1
```
