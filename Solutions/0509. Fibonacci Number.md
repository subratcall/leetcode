## 解法2: Top-Down Approach using Memoization

TC：O(n) SC: O(n)
```py
class Solution:
    cache = {0: 0, 1: 1}
    def fib(self, n: int) -> int:
        if n in self.cache:
            return self.cache[n]
        self.cache[n] = self.fib(n-1) + self.fib(n-2)
        return self.cache[n]
```
