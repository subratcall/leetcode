## 解法：


```
class Solution:
    def findMin(self, nums: List[int]) -> int:
        left, right = 0, len(nums) - 1
        while left + 1 < right:
            mid = (left + right) // 2
            if nums[mid] < nums[-1]:
                right = mid
            elif nums[mid] > nums[-1]:
                left = mid
            else:
                right = mid
        return min(nums[left], nums[right])
```
