优先满足饥饿度最小的孩子，且把能满足他的最小饼干给他。

这样就需要对input进行排序，来减少时间复杂度。

TC: O(NlogN) + O(MlogM)

SC: O(1)
```
class Solution:
    def findContentChildren(self, g: List[int], s: List[int]) -> int:
        g.sort()
        s.sort()
        res = 0
        
        i, j = 0, 0
        while i < len(g) and j < len(s):
            if g[i] <= s[j]:
                res += 1
                i += 1
            j += 1
        
        return res
```
