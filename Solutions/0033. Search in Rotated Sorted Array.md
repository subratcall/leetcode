## 解法1: 用两次二分

先用一次二分，找到最小值和它的index。这样就能确定target是在左半边还是右半边。

然后再对那半边进行二分搜索。

逻辑清晰，也不容易写错。
```
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        min_idx = self.find_min(nums, target)
        if nums[min_idx] <= target <= nums[-1]:
            return self.binary_search(nums, min_idx, len(nums) - 1, target)
        return  self.binary_search(nums, 0, min_idx, target)
    
    def find_min(self, nums, target):
        left, right = 0, len(nums) - 1
        while left + 1 < right:
            mid = (left + right) // 2
            if nums[mid] > nums[-1]:
                left = mid
            else:
                right = mid
        return left if nums[left] < nums[right] else right
    
    def binary_search(self, nums, left, right, target):
        while left + 1 < right:
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid
            elif nums[mid] > target:
                right = mid
            else:
                return mid
        if nums[left] == target:    return left
        if nums[right] == target:   return right
        return -1
```

## 解法2: 一次二分
在中间切一刀。先判断这一刀切在了左半段，还是右半段。

对于每一种情况，再判断target应该往左边找，还是往右边找。
```
class Solution:
    def search(self, A: List[int], target: int) -> int:
        left, right = 0, len(A) - 1
        while left + 1 < right:
            mid = (left + right) // 2
            if A[mid] >= A[left]:  # 切在了左边
                if A[left] <= target <= A[mid]:
                    right = mid
                else:
                    left = mid
            else:  # 切在了右边
                if A[mid] <= target <= A[right]:
                    left = mid
                else:
                    right = mid
        if A[left] == target:  return left 
        if A[right] == target:  return right
        return -1
```
