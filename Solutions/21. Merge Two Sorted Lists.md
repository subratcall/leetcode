## 解法1: 递归
先找到结果的头（是l1还l2），然后返回缩小规模后的问题的答案。再把结果和小答案合并得到大答案。

TC：O(m+n) SC:O(m+n)

```
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeTwoLists(self, l1: ListNode, l2: ListNode) -> ListNode:
        #递归出口
        if not l1:
            return l2
        if not l2: 
            return l1
        #递归拆解
        if l1.val < l2.val:
            l1.next = self.mergeTwoLists(l1.next, l2)
            return l1
        else:
            l2.next = self.mergeTwoLists(l1, l2.next)
            return l2
```

## 解法2: 迭代

TC：O(m+n) SC:O(1)

```
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def mergeTwoLists(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        if not l1:
            return l2
        if not l2:
            return l1
        
        dummy = ListNode()
        p = dummy
        
        while l1 and l2:
            if l1.val < l2.val:
                p.next = l1
                l1 = l1.next
            else:
                p.next = l2
                l2 = l2.next
            p = p.next
        
        if l1:
            p.next = l1
        if l2:
            p.next = l2
            
        return dummy.next
```
